## Upgrade Instructions

Follow these steps to update SHR Creative Studio to the latest version.

<Steps>
  <Step title="Backup Your Project" icon="database">

  Create a backup of your current project directory.

  ```
  cp -r your-project ~/backup-$(date +%Y%m%d)
  ```

  </Step>
  <Step title="Update Package" icon="package">
    <Tabs>
      <Tab title="npm" icon="npm">
````bash
npm install shr-creative-studio@latest
````
      </Tab>
      <Tab title="yarn" icon="yarn">
````bash
yarn add shr-creative-studio@latest
````
      </Tab>
    </Tabs>
  </Step>
  <Step title="Run Migration" icon="settings">
    <CodeGroup tabs="CLI,Script">
````bash
npx shr-migrate config.yaml
````
````javascript
// migration.js
const { migrateConfig } = require('shr-creative-studio/migrate');
migrateConfig('config.yaml', 'config.json');
````
    </CodeGroup>
  </Step>
  <Step title="Restart Server" icon="refresh-cw">
    Restart your development server and verify changes.
  </Step>
</Steps>

<Callout kind="tip">
  After upgrading, check the [Quickstart](/quickstart) guide for any new configuration options.
</Callout>
