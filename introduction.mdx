## Overview

SHR Creative Studio provides a centralized documentation space where you organize and manage all your project documentation. Create structured pages, embed multimedia, and collaborate seamlessly with your team. Use it to document APIs, user guides, or creative workflows with full customization.

<Callout kind="info">
SHR Creative Studio uses brand color `#3B82F6` for primary elements. Apply it consistently in your custom themes.
</Callout>

## Quick Start

Get up and running in minutes. Follow these steps to create your first documentation space.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.shrcreative.studio/register` with your email.
  </Step>
  <Step title="New Space" icon="plus">
    Click "New Space" and name it, e.g., "Project Docs".
  </Step>
  <Step title="Add Page" icon="file-plus">
    Create your first page and start editing with markdown or the visual builder.
  </Step>
  <Step title="Publish" icon="globe">
    Share your space via public link or embed code.
  </Step>
</Steps>

<CodeGroup tabs="Markdown,HTML">
```markdown
# Welcome Page

Welcome to SHR Creative Studio.

## Quick Links

- [Quickstart](/quickstart)
- [Guides](/guides)
```
```html
<!DOCTYPE html>
<html>
<head>
  <title>SHR Creative Studio</title>
  <style>body { color: #3B82F6; }</style>
</head>
<body>
  <h1>Welcome</h1>
  <p>Start documenting.</p>
</body>
</html>
```
</CodeGroup>

## Brand Guidelines

Adhere to these guidelines for a cohesive look.

<Expandable title="Primary Brand Color" default-open="true">
Use `#3B82F6` for buttons, links, and headers. Avoid variations unless specified.

**Examples:**
- Background: `background-color: #3B82F6;`
- Text: `color: #3B82F6;`

<Callout kind="tip">
Test colors for accessibility. Ensure contrast ratios exceed 4.5:1.
</Callout>
</Expandable>

<Expandable title="Typography">
Prefer sans-serif fonts like Inter or system defaults. Headings: bold, 1.2em line-height.
</Expandable>
